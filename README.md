# Marketing-Funnel-Conversion-Analysis
## Project Overview
- This project analyzes a digital marketing funnel to understand user drop-off, conversion performance behhavior across traffice sources and devices. The goal is to identify optimization opportunities that improve conversion rates and revenue outcomes.
## Dataset
 - Source: Simulated marketing funnel dataset
 - Key Fields:
    - User_ID
    - Funnel_Stage (Visit, View, Add to Cart, Checkout, Purchase)
    - Traffic_Source
    - Device
    - Revenue
## Funnel Structure
- The funnel dollows a standard e-commerce journey:
  - Visit 
  - View 
  - Add to Cart 
  - Checkout 
  - Purchase
- Distinct users were counted at each stage to avoid double-counting repeat actions.

## Key Insights
  - The largest drop-off occurs between Checkout and Purchase, where nearly 48% of users abandon.
  - This suggests potential friction during payment, pricing, or checkout UX.
  - Mobile and Desktop show comparable purchase volumes, indicating consistent cross-device performance.
  - Certain taffic sources drive a disproportionate share of purchases, highlighting strong bottom0funnel intent.
